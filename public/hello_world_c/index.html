Hello, World! на Си

06 January 2025

Короткий кусок кода из учебника

“Программирование на С для начинающих”

Пример вывода простого текста в консоль с помощью printf()

#include <stdio.h>
main()
{
    printf("Hello, World!");
    return 0;
}

Интересно что делают <stdio.h> include и return 0 тем более возвращает 0.
